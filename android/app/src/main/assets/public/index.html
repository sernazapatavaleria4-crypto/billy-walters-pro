📚 Guía Kelly Criterion
Formula: f* = (b·p - q) / b donde b = cuota-1, p = probabilidad, q = 1-p
Billy Walters usaba Kelly Fraccionado al 30% para reducir la varianza sin sacrificar mucho del crecimiento óptimo.
Regla de oro: Nunca apostar más del 12% del bankroll en un solo juego, sin importar el edge calculado. El Kelly Full puede llevar a ruina del bankroll incluso con edge real — siempre usa la versión fraccionada.
